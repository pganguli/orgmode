p = paste below
P = paste above

"{CHAR} = append to register /CHAR/
"" = cut/copyied text
"* = primary selection
"+ = secondary selection

q{char} = record macro in register /char/
@{char} = replay macro from register /char/

m{char} = set local mark in register /char/
m{CHAR} = set global mark in register /CHAR/
' = jump to marked line
` = jump to marked character

/ = search. Use \<word\> to match whole `word`
* = search for word under cursor

                                          and replace it with <replace>; all occurrences
                                          if <flags> contains g; ignore case if <flags> contains i;
                                          strict case if <flags> contains I.
